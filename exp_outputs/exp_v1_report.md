## 1. 实验概述

### 1.1 目的

本实验是 T20 选题"多传感器贝叶斯不确定性融合——冲突损害的层次模型"的**实验版本 v1**。目标是用 2 种遥感传感器（VIIRS 夜间灯光 + Sentinel-2 多光谱），对马里乌波尔战区进行损害概率估计，并通过贝叶斯层次模型和 Dempster-Shafer 证据理论进行融合，比较不同融合策略的效果。

### 1.2 v1 命名约定

所有输出文件均带 `exp_v1_` 前缀，表示这是实验第一版。这不是可发表的最终结果——它存在的意义是：

1. 跑通完整方法链，验证技术可行性
2. 暴露数据和方法中的问题
3. 作为学习和迭代的起点

### 1.3 核心结论（TL;DR）

- **VIIRS 和 Sentinel-2 在此实验中几乎完全不一致**（相关性 ρ=0.04）
  - VIIRS：97% 的像素被判断为"正常"（p₁ ≈ 0.02），只有约 3% 显示显著灯光下降
  - Sentinel-2：约 50% 的像素被判断为"损害"（p₂ ≈ 0.51），几乎均匀分布
- **根因**：Sentinel-2 数据存在严重的季节混淆（1月冬季 → 5月春末，NDVI 上升 0.14），光学变化中季节信号完全压倒了冲突信号
- **贝叶斯融合**在传感器分歧时给出了合理的行为：后验收缩到 0.38，介于两者之间
- **D-S 融合在本实验中退化**：因为 Bootstrap 估计的传感器不确定性 σ 过低，导致 D-S 没有"无知区间"


### 3.2 贝叶斯层次模型详解

```
Level 3 (全局先验):
    p_trueᵢ ~ Beta(α₀, β₀)
    实验中: α₀=1, β₀=1 → 均匀分布 Uniform(0,1)

Level 2 (传感器观测):
    pₖ|p_true ~ Beta(κₖ·p_true, κₖ·(1-p_true))
    κₖ = 传感器精度, 由 1/σ²ₖ 决定
    κ 越大 → 传感器越"可靠" → 后验更靠近该传感器的估计

Level 1 (后验推断):
    p_true | p₁,p₂ ~ Beta(α₀ + κ₁p₁ + κ₂p₂, β₀ + κ₁(1-p₁) + κ₂(1-p₂))
    → 后验均值 = (α₀+κ₁p₁+κ₂p₂) / (α₀+β₀+κ₁+κ₂)
    → 后验方差 = αβ / [(α+β)²(α+β+1)]
```

**物理直觉**：
- 两个传感器一致时 → 后验 = 单传感器估计（方差减小）
- 两个传感器分歧时 → 后验在两者之间（方差增大）
- κ 值大的传感器"话语权"更大

### 3.3 Dempster-Shafer 证据理论详解

```
识别框架: Θ = {D (损害), U (未损害)}

基本概率分配 (BPA):
    mₖ({D})  = pₖ · (1 - σₖ_norm)    ← 支持"损害"的证据
    mₖ({U})  = (1-pₖ) · (1 - σₖ_norm) ← 支持"未损害"的证据  
    mₖ(Θ)    = σₖ_norm               ← "不确定"的证据

Dempster 组合规则:
    冲突: K = m₁({D})·m₂({U}) + m₁({U})·m₂({D})
    m₁₂({D}) = [m₁(D)·m₂(D) + m₁(D)·m₂(Θ) + m₁(Θ)·m₂(D)] / (1-K)
    m₁₂({U}) = [m₁(U)·m₂(U) + m₁(U)·m₂(Θ) + m₁(Θ)·m₂(U)] / (1-K)
    m₁₂(Θ)   = m₁(Θ)·m₂(Θ) / (1-K)

输出:
    Bel(损害) = m₁₂({D})              ← 信任下界
    Pl(损害)  = m₁₂({D}) + m₁₂(Θ)    ← 信任上界
    不确定性  = Pl - Bel = m₁₂(Θ)     ← 认知不确定性
```

## 5. 每一步的输入输出

### Step 1: 数据预处理

**输入：**
- `VNP46A2.A2022008.h21v04.002.2025114204956.h5` (VIIRS 战前, ~28 MB)
- `VNP46A2.A2022128.h21v04.002.2025119063131.h5` (VIIRS 战后, ~29 MB)
- `战前1/`, `战前2/`, `战后1/`, `战后2/` (各约 1.5 GB Sentinel-2 数据)

**做了什么：**
1. 读取 VIIRS HDF5 的 `DNB_BRDF-Corrected_NTL` 波段
2. 应用质量掩膜 (`Mandatory_Quality_Flag ≤ 1`)
3. 计算 ΔNTL = 战后 - 战前
4. 根据经纬度裁剪马里乌波尔子区域 (46.85°-47.86°N, 37.66°-39.14°E)
5. 对每个 Sentinel-2 tile，读取 10m 波段计算 NDVI 和 NDWI，读取 20m 波段计算 NBR 和 NDBI
6. 计算战前-战后差值: ΔNDVI, ΔNDWI, ΔNBR, ΔNDBI
7. 从 SCL 图层提取有效像素掩膜

**输出：**

| 文件 | 大小 | 形状 | 含义 |
|------|------|------|------|
| `exp_v1_viirs_dntl.npy` | 0.3 MB | (242,355) | VIIRS ΔNTL |
| `exp_v1_viirs_lat.npy` | 2 KB | (242,) | 纬度坐标 |
| `exp_v1_viirs_lon.npy` | 3 KB | (355,) | 经度坐标 |
| `exp_v1_s2_dNDVI_t37tdn.npy` | 470 MB | (10980,10980) | T37TDN ΔNDVI |
| `exp_v1_s2_dNDWI_t37tdn.npy` | 470 MB | (10980,10980) | T37TDN ΔNDWI |
| `exp_v1_s2_dNBR_t37tdn.npy` | 118 MB | (5490,5490) | T37TDN ΔNBR |
| `exp_v1_s2_dNDBI_t37tdn.npy` | 118 MB | (5490,5490) | T37TDN ΔNDBI |
| `exp_v1_s2_mask_t37tdn.npy` | 29 MB | (5490,5490) | 有效像素掩膜 |
| （T37TCN 同上 5 个文件） | | | |
| **合计** | **~2.5 GB** | | |

### Step 2: 单传感器概率估计

**输入：** Step 1 的所有输出

**做了什么：**

*VIIRS 部分：*
1. 提取非 NaN 的 ΔNTL 像素值
2. 用 2 分量 Gaussian Mixture Model (GMM) 拟合
3. 识别"正常波动"分量（均值接近 0）和"显著下降"分量（均值更负）
4. 每个像素计算属于"损害"分量的后验概率 → p₁
5. Bootstrap 200 次重采样 → 估计 GMM 参数的不确定性 → σ²₁

*Sentinel-2 部分：*
1. 对每个 tile 的 87M+ 有效像素，提取 4 维特征向量 (ΔNDVI, ΔNDWI, ΔNBR, ΔNDBI)
2. 标准化 + PCA → PC1 解释 94%+ 的方差
3. Hotelling T² 检验 (H₀: 多元均值为 0)
4. sigmoid 映射 PC1 到 [0,1] → p₂
5. Bootstrap 50 次 (因为数据量太大) → σ²₂

*空间对齐：*
- S2 10m 分辨率上的 p₂ → 50倍块平均 → ~500m 网格
- 裁剪到与 VIIRS 子区域一致

**输出：**

| 文件 | 形状 | 含义 |
|------|------|------|
| `exp_v1_p1_viirs.npy` | (242,355) | VIIRS 原始网格上 p₁ |
| `exp_v1_s1_viirs.npy` | (242,355) | VIIRS σ₁ |
| `exp_v1_p1_unified.npy` | (219,219) | 统一网格上 p₁ |
| `exp_v1_s1_unified.npy` | (219,219) | 统一网格上 σ₁ |
| `exp_v1_p2_unified.npy` | (219,219) | 统一网格上 p₂ |
| `exp_v1_s2_unified.npy` | (219,219) | 统一网格上 σ₂ |
| `exp_v1_grid_info.npz` | — | H=219, W=219, pixel=500m |

### Step 3: 贝叶斯层次融合

**输入：** `p1_unified, s1_unified, p2_unified, s2_unified`

**做了什么：**
1. 提取有效像素（两个传感器都有值的）
2. 计算传感器精度 κₖ = clamp(τ/σ²ₖ, 1, 50)
3. 用 Beta-Binomial 共轭解析计算后验：
   - α_post = α₀ + κ₁p₁ + κ₂p₂
   - β_post = β₀ + κ₁(1-p₁) + κ₂(1-p₂)
   - p_fused = α/(α+β), σ_fused = sqrt(αβ/((α+β)²(α+β+1)))
4. 计算每个像素的 95% HDI（Beta 分位数）

**为什么用解析近似而非 MCMC：**
- Windows 环境无 C++ 编译器，PyMC MCMC 在纯 Python 模式下极慢
- 对于 Beta-Binomial 层次模型，共轭解析解是标准且精确的
- 完整 MCMC 实现保留给后续版本（在 Linux/Mac 或有 C 编译器的环境下）

**输出：**

| 文件 | 含义 |
|------|------|
| `exp_v1_bayes_fused_mean.npy` | 融合后验均值 p_fused |
| `exp_v1_bayes_fused_std.npy` | 融合后验标准差 σ_fused |
| `exp_v1_bayes_fused_hdi_low.npy` | 95% HDI 下界 |
| `exp_v1_bayes_fused_hdi_high.npy` | 95% HDI 上界 |

### Step 4: D-S 证据理论融合

**输入：** 同 Step 3

**做了什么：**
1. 将 σₖ 归一化为 [0, 1] → 作为"无知度" mₖ(Θ)
2. 构建 BPA: m({D}) = p·(1-σ), m({U}) = (1-p)·(1-σ)
3. Dempster 组合规则（矢量化逐像素计算）
4. 额外计算简单投票和加权投票基线

**输出：**

| 文件 | 含义 |
|------|------|
| `exp_v1_ds_bel.npy` | Bel(损害) — 信任下界 |
| `exp_v1_ds_pl.npy` | Pl(损害) — 信任上界 |
| `exp_v1_ds_ignorance.npy` | Pl - Bel — 不确定性区间 |
| `exp_v1_ds_conflict.npy` | 冲突质量 K |
| `exp_v1_vote_mean.npy` | (p₁+p₂)/2 基线 |
| `exp_v1_vote_weighted.npy` | 精度加权基线 |

### Step 5: 评估与可视化

**输入：** Step 2-4 的所有输出

**做了什么：**
1. 生成 6 面板对比图 (VIIRS | S2 | Bayes | Bayes-σ | D-S | Vote)
2. 生成散点矩阵 (p₁ vs p₂, p₁ vs Bayes, p₂ vs Bayes)
3. 合成验证实验：注入 100 个已知损害像素（3种场景），检验融合能恢复到什么程度
4. 计算并保存数值指标

## 6. 实验结果

### 6.1 单传感器概率统计

```
传感器          均值    标准差   P10    P50    P90
─────────────────────────────────────────────────
VIIRS (GMM)    0.018   0.128   0.000  0.000  0.001
Sentinel-2     0.513   0.248   0.164  0.514  0.823
```

**解读：**
- VIIRS 几乎把所有像素都判断为"正常"（97% 在正常分量中）
  - GMM 正常分量均值 = +0.155（略微变亮），损害分量均值 = -2.120（明显变暗）
  - 损害分量只占 3.1% 的权重
- Sentinel-2 在 0-1 之间几乎均匀分布，均值 0.51
  - 这不是"一半像素损害了"，而是季节变化导致 S2 变化指数分布接近均匀

### 6.2 融合结果

```
方法               均值    标准差   P10    P50    P90
─────────────────────────────────────────────────────
VIIRS-only        0.018   0.128   0.000  0.000  0.001
S2-only           0.513   0.248   0.164  0.514  0.823
Bayesian Fusion   0.383   0.071   0.292  0.380  0.458
D-S Bel           0.053   0.117   0.003  0.018  0.103
Vote Mean         0.266   0.142   0.083  0.261  0.416
Vote Weighted     0.263   0.136   0.083  0.260  0.415
```

### 6.3 关键指标

| 指标 | 值 | 含义 |
|------|-----|------|
| 传感器相关性 ρ(p₁,p₂) | 0.041 | 几乎完全不相关 |
| 像素分歧比例 | 51.3% | 一半以上的像素两传感器判断相反 |
| 贝叶斯 vs 投票差异 | 0.122 | 融合偏离简单平均 |

### 6.4 合成验证

```
方法            AUC
──────────────────────
VIIRS-only      0.980   ← 对VIIRS主导场景表现最好
S2-only         0.743   ← 对S2主导场景表现中等
Vote Mean       0.983   ← 简单平均意外地好
Bayesian        0.983   ← 与投票相当
```

合成实验中贝叶斯融合和简单投票性能相当，因为在 κ₁=κ₂ 的情况下两者等价。如果传感器精度不同（κ₁≠κ₂），贝叶斯融合的优势才会体现。

## 7. 关键发现与讨论

### 7.1 传感器几乎完全不相关——为什么？

**ρ(p₁, p₂) = 0.041** 意味着两个传感器在判断"哪些像素被损害"时几乎没有共识。这很惊人，但原因清晰：

1. **VIIRS 检测的是灯光消失** → 断电、人口撤离。马里乌波尔只有少量像素 NTL 显著下降。
2. **Sentinel-2 检测的是地表反射率变化** → 但在 1月→5月期间，90%+ 的变化来自植被季节生长。

**这不是方法的问题，是数据的问题。** 两个传感器在检测完全不同的东西，且 S2 的信号被季节效应污染了。

### 7.2 Bootstrap σ 估计严重偏低

Bootstrap 重采样估计的传感器不确定性 σ 极小（VIIRS σ₁ ≈ 0.006，S2 σ₂ ≈ 0.000）。原因是：
- GMM 模型在 76K 样本上非常稳定 → Bootstrap 方差极小
- PCA 在 87M 样本上几乎不变 → Bootstrap 方差为零（数值精度）

**这导致**：
- 贝叶斯融合中 κₖ 都饱和为相同的值 → 退化为简单平均
- D-S 融合中 m(Θ) ≈ 0 → 没有"不确定"分量 → 退化

**解决方案**（见第 9 节）：
- 用预测不确定性（prediction uncertainty）而非参数不确定性
- 校准 σ（如 Platt scaling）

### 7.3 贝叶斯融合的合理行为

尽管传感器分歧严重，贝叶斯融合展示了合理的行为：
- **收缩 (shrinkage)**：后验从两个极端 (0.02, 0.51) 收缩到中间值 0.38
- **高不确定性**：后验 σ ≈ 0.21，95% HDI 宽度 ≈ 0.78 —— 这正确地反映了"我们不知道"
- **先验影响可解释**：Beta(1,1) 先验 = 均匀分布，当两个传感器不可靠时，融合退向先验（均匀）

### 7.4 D-S 在本实验中的退化

D-S 融合在本实验中表现不佳：
- Bel(D) ≈ 0.05, 几乎为 0
- Pl-Bel ≈ 0，没有任何认知不确定性区间

这是因为 m₂(Θ) = σ₂_norm ≈ 0（S2 声称自己 100% 确定），而 m₁ 虽然有一些不确定（m₁(Θ) ≈ 0.05），但它同时又声称 m₁(U) = 0.95（几乎确定"无损害"）。在 D-S 框架下，VIIRS 对"未损害"的强烈证据压过了 S2 对"损害"的证据，导致融合结果偏向"未损害"。

**教训：D-S 对 BPA 的构建方式极其敏感。** σ 的估计质量直接决定了 D-S 融合的有效性。

## 8. 已知问题与局限

### 8.1 致命问题

| 问题 | 严重程度 | 原因 |
|------|---------|------|
| S2 季节混淆 | 🔴 致命 | Jan(冬季) vs May(春季)，ΔNDVI=+0.14，信号完全被季节主导 |
| 无地面真值 | 🔴 致命 | 所有验证都是软验证或合成验证 |
| σ 估计偏差 | 🟡 中等 | Bootstrap on model params ≠ prediction uncertainty |

### 8.2 次要问题

- **S2 分辨率不一致**：NBR/NDBI 在 20m，NDVI/NDWI 在 10m → 上采样引入伪影
- **空间对齐粗糙**：块平均降采样忽略地理坐标精确匹配
- **PyMC MCMC 不可用**：Windows 无 C 编译器 → 只能用解析近似
- **双 tile 未镶嵌**：v1 仅用了 T37TDN 数据，T37TCN 被丢弃
